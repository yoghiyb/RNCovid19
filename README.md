# RNCovid19

Aplikasi untuk moitoring perkembangan virus covid19 di indonesia menggunakan React Native

## How to install :

```bash
    git clone https://github.com/yoghiyb/RNCovid19.git

    cd RNCovid19

    npm install

    npx react-native run-android
```
